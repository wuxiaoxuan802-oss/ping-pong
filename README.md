# 🏓 手勢乒乓球 (Gesture Ping Pong)


這是一個基於 **MediaPipe** 與 **Pygame** 開發的創新體感遊戲。玩家無需滑鼠或鍵盤，只需伸出食指即可即時控制球拍，透過捏合手勢抓取與發射球，挑戰不斷加速的球與各種奇特的怪獸障礙物。

---

## ✨ 遊戲特色

- **手部與手勢追蹤**: 使用 MediaPipe Task API 進行極低延遲的食指指尖追蹤。支援「✌️ (Victory) 手勢」快速開局，以及「捏合」手指抓取與發射球的物理機制。
- **動態難度與幀率獨立**: 隨關卡提升，球速與怪獸的行列數會自動增加。採用 `time_scale` 運算，確保在不同硬體效能與幀率下，球的移動速度保持一致且流暢。
- **自動拍照系統**: 遊戲中會隨機捕捉您的遊玩瞬間，並在遊戲結束 (Game Over) 畫面以「拍立得」風格展示回憶！
- **沉浸式聲光效果**: 支援自訂背景音樂 (BGM) 自動播放，並使用 NumPy 即時合成多種合成器音效 (打擊、破壞、漏球、升級)，搭配粒子爆破與震動特效，帶來極致科幻感。
- **豐富技能道具**: 擊碎怪獸時有機會掉落多達 7 種各具特色的技能道具，改變戰局。

## 🎮 操作說明

1. **控制**: 將食指指尖對準攝影機，水平移動即可控制球拍位置。
2. **抓取與發射**: 當球落下時，**將食指與拇指捏合**即可抓取球，隨後快速揮動手臂並鬆開手指，即可將球拋出。
3. **開始/重啟**: 對著鏡頭比出 **✌️ (Victory)** 手勢，或按下 `[空白鍵]`。
4. **退出**: 按下 `[Q]` 或 `[ESC]` 鍵。

## 🚀 快速啟動

### 1. 安裝環境 (推薦使用 uv)

本專案全面支援 `uv` 進行套件管理。請開啟終端機並執行：

```powershell
uv add pygame opencv-python mediapipe numpy
```

### 2. 執行遊戲

使用 `uv run` 來啟動您的遊戲：

```powershell
uv run ping_pong.py
```

## 📂 專案結構

- `ping_pong.py`: 遊戲主程式邏輯，包含物理引擎、渲染與手勢識別。
- `msjh.ttc`: 支援中文顯示的字型檔。
- `hand_landmarker.task`: MediaPipe 手部偵測模型。
- `pyproject.toml`: 專案與依賴管理設定檔 (由 uv 管理)。
