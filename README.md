# 🏓 手勢乒乓球 (Gesture Ping Pong)

![Hero Image](assets/hero.png)

這是一個基於 **MediaPipe** 與 **Pygame** 開發的創新體感遊戲。玩家無需滑鼠或鍵盤，只需伸出食指即可即時控制球拍，挑戰不斷加速的球與各種奇特的怪獸障礙物。

---

## ✨ 遊戲特色
- **手部追蹤**: 使用 MediaPipe Task API 進行極低延遲的食指指尖追蹤。
- **霓虹美學**: 沉浸式的科幻視覺效果，包含粒子爆破、動態星空背景。
- **動態難度**: 隨關卡提升，球速與怪獸數量會自動調整。
- **豐富技能**: 多達 7 種各具特色的技能道具，改變戰局。

## 🎮 操作說明
1. **控制**: 將食指指尖對準攝影機，水平移動控制球拍。
2. **開始/重啟**: 比出 **✌️ (Victory)** 手勢或按下 `[空白鍵]`。
3. **發球**: 在抓到球後快速揮動手部即可將球拋出。
4. **退出**: 按下 `[Q]` 或 `[ESC]` 鍵。

## 🛠️ 技能道具圖鑑
| 符號 | 名稱 | 效果說明 |
| :---: | :--- | :--- |
| **B** | 巨球 | 球體變大，大幅提升接球機率。 |
| **P** | 長板 | 球拍寬度延長 1.5 倍。 |
| **S** | 減速 | 球速永久降低 30%（僅限當前回合）。 |
| **M** | 分裂球 | 產生 2 顆額外的能量球。 |
| **2** | 雙倍得分 | 限時內所有得分乘以二。 |
| **H** | 護盾 | 底部產生防護牆，自動反彈漏掉的球。 |
| **!** | 加速衝刺 | 球速提升 40%，難度倍增。 |

## 🚀 快速啟動

### 1. 安裝環境 (推薦使用 uv)
```powershell
uv add pygame opencv-python mediapipe numpy
```

### 2. 執行遊戲
```powershell
uv run ping_pong.py
```

### 3. 查看主頁
直接開啟專案目錄下的 `index.html` 即可查看精美的視覺介紹頁。

---

## 📂 專案結構
- `ping_pong.py`: 遊戲主程式邏輯。
- `index.html`: 專案網頁主頁 (Landing Page)。
- `assets/`: 儲存遊戲截圖與視覺資源。
- `msjh.ttc`: 支援中文顯示的字型檔。
- `hand_landmarker.task`: MediaPipe 手部偵測模型。
